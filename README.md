# 반응형 메모 앱 (Responsive Note-taking App)

Create React App을 사용하여 구축된 반응형 메모 앱입니다.

## 주요 기능

1. **회원가입**: 사용자명, 이메일, 비밀번호로 새 계정을 생성할 수 있습니다.
2. **로그인**: 등록된 이메일과 비밀번호로 로그인할 수 있습니다.
3. **로그아웃**: 헤더의 로그아웃 버튼을 통해 로그아웃할 수 있습니다.
4. **새 노트 작성**: "+ 새 노트" 버튼을 클릭하여 새로운 노트를 생성할 수 있습니다.
5. **노트 수정**: 노트를 선택하고 제목과 내용을 수정할 수 있습니다.
6. **노트 저장 및 삭제**: 모든 노트는 자동으로 로컬 스토리지에 저장되며, 삭제 버튼을 통해 노트를 삭제할 수 있습니다.
7. **검색 기능**: 제목 또는 내용으로 노트를 검색할 수 있습니다.
8. **사용자별 노트 분리**: 각 사용자는 자신만의 노트를 관리할 수 있습니다.

## 설치 및 실행

### 필수 요구사항
- Node.js (v14 이상)
- npm 또는 yarn

### 설치
```bash
cd memo-app
npm install
```

### 개발 서버 실행

**방법 1: 브라우저 자동 열림 (권장)**
```bash
npm run start:open
```
또는 Windows에서:
```bash
start.bat
```

**방법 2: 기본 실행**
```bash
npm start
```
브라우저가 자동으로 열리지 않으면 수동으로 [http://localhost:3000](http://localhost:3000)을 열어주세요.

### 프로덕션 빌드
```bash
npm run build
```

### 웹서버 배포

**가장 빠른 방법: Netlify 드래그 앤 드롭**
1. https://app.netlify.com/drop 접속
2. `build` 폴더를 드래그 앤 드롭
3. 배포 완료! 자동으로 URL이 생성됩니다.

**자세한 배포 가이드**: [DEPLOYMENT.md](./DEPLOYMENT.md) 또는 [quick-deploy.md](./quick-deploy.md) 참조

**배포 스크립트 실행** (Windows):
```bash
deploy.bat
```

## 반응형 디자인

이 앱은 다양한 화면 크기에 대응하는 반응형 디자인을 구현했습니다:

- **데스크톱 (> 768px)**: 사이드바와 메인 콘텐츠 영역이 나란히 표시됩니다.
- **태블릿/모바일 (≤ 768px)**: 사이드바가 상단에 위치하고, 메인 콘텐츠가 하단에 표시됩니다.
- **작은 모바일 (≤ 480px)**: 더 작은 폰트 크기와 패딩으로 최적화됩니다.

## 데이터 저장

- **사용자 정보**: 모든 사용자 정보는 로컬 스토리지에 저장됩니다.
- **사용자별 노트**: 각 사용자의 노트는 사용자 ID를 기반으로 분리되어 저장됩니다.
- **자동 저장**: 노트는 자동으로 로컬 스토리지에 저장됩니다. 페이지를 새로고침하거나 브라우저를 닫았다가 다시 열어도 데이터가 유지됩니다.

## 사용 방법

1. **첫 사용 시**: 회원가입 화면에서 새 계정을 생성합니다.
2. **로그인**: 이메일과 비밀번호로 로그인합니다.
3. **노트 관리**: 로그인 후 노트를 작성, 수정, 삭제할 수 있습니다.
4. **로그아웃**: 작업을 마친 후 로그아웃합니다.

## 보안 참고사항

⚠️ **중요**: 현재 버전은 데모/개발 목적으로 제작되었습니다. 비밀번호가 평문으로 저장되므로, 프로덕션 환경에서는 반드시 비밀번호 해시화 및 백엔드 서버 구축이 필요합니다.
